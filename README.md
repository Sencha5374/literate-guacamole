# Калькулятор на Python

## Описание
Простой консольный калькулятор, разработанный для практической работы с системой контроля версий Git. Программа позволяет выполнять базовые арифметические операции и демонстрирует основы написания чистого кода на языке Python.

## Функционал
Программа поддерживает следующие операции:
- **Сложение** (+)
- **Вычитание** (-)
- **Умножение** (*)
- **Деление** (/) с защитой от деления на ноль

Пользователь может последовательно выполнять вычисления через интерактивное меню, выбирая нужную операцию и вводя числа.

## Как установить и запустить

### Требования
- Python 3.x (рекомендуется версия 3.6 и выше)

### Установка
1. Клонируйте репозиторий:
   ```bash
   git clone <адрес_вашего_репозитория>
   ```
2. Перейдите в папку проекта:
   ```bash
   cd <имя_папки_репозитория>
   ```

### Запуск
Запустите программу командой:
```bash
python calculator.py
```
или
```bash
python3 calculator.py
```

Следуйте инструкциям в консоли для выполнения вычислений.
